Welcome To Our
Application

EasyTapp Mobile adalah aplikasi absensi berbasis Android yang memudahkan admin dalam mengelola data kehadiran karyawan secara fleksibel dan real-time langsung dari smartphone. Melalui fitur manajemen karyawan, pemantauan absensi, serta integrasi RFID Tag sebagai identitas digital, aplikasi ini memberikan solusi modern dan efisien untuk meningkatkan akurasi dan produktivitas Sistem Absensi.

About Our Application

EasyTapp merupakan hasil kolaborasi antara mahasiswa semester 6 Program Studi Teknik Multimedia dan Jaringan (TMJ) Politeknik Negeri Jakarta dengan PT Bithouse, yang bertujuan mengintegrasikan pengetahuan akademik dengan kebutuhan industri melalui pengembangan aplikasi absensi digital berbasis teknologi terkini.

Application
Advantages

01
Embeded with OTP to send reset password confirmation using API

Our application integrates One-Time Password (OTP) technology to ensure secure password reset processes. This reduces the risk of unauthorized access and enhances user trust by verifying identity through a unique code sent to the registered email or phone number.

02
Monitor Real time attendance with Image validation

This feature is especially valuable for organizations requiring strict identity validation, reducing the chance of proxy check-ins and enabling transparent employee tracking.

03
Embeded with IOT SYSTEM that store data when offline

The system is embedded with IoT capabilities that allow it to store attendance and user data even when there is no internet connection. Once the connection is restored, all stored data is synced automatically to ensure no records are lost.

04
Secure with RLS Supabase and AUTH

Built with Supabase Row-Level Security (RLS) and modern authentication systems, the application ensures data privacy and secure access management. This guarantees that users only see the data theyâ€™re authorized to access.
